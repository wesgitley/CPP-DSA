# C++ Data Structures and Algorithms (DSA)

This repository contains implementations of various data structures and algorithms in C++ from numbered exercises (One.c to Seven.c).

## Project Structure

- `main.cpp`: Main entry point.
- `One.c` to `Seven.c`: Individual C++ files with DSA examples.
- `C++ Programming in IoT.cbp`: Code::Blocks project file.

## Prerequisites

- Code::Blocks IDE or g++ compiler.

## Build and Run

### Using Code::Blocks

1. Open `C++ Programming in IoT.cbp`.
2. Build → Build and Run (F9).

### Using g++ (Command Line)

```
g++ *.cpp *.c -o CPP-DSA.exe
./CPP-DSA.exe
```
